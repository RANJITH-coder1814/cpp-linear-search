# C++ GCD of Two Numbers

## 📌 Overview

This project demonstrates how to calculate the **Greatest Common Divisor (GCD)** of two numbers using the **Euclidean Algorithm** in C++.
The GCD of two integers is the largest positive integer that divides both numbers without leaving a remainder.

## 🚀 Features

* Simple and beginner-friendly C++ code
* Uses the efficient **Euclidean Algorithm**
* Accepts user input
* Displays the GCD result

## 🧠 Algorithm

The Euclidean Algorithm works by repeatedly applying:

a % b

until **b becomes 0**.
The remaining value of **a** will be the GCD.

## 💻 Example

Input

```
Enter two numbers: 24 36
```

Output

```
GCD of 24 and 36 is: 12
```

## 🛠️ How to Run

### Compile

```
g++ gcd.cpp -o gcd
```

### Run

```
./gcd
```

## ⏱️ Time Complexity

* **O(log(min(a, b)))**
